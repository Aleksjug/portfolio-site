What I Do

Web Development

Over the past two years I competed in two separate Hackathons through Queen’s University that both involved creating web-apps using React. The first project I gained experience using the MapBox API to present geographical COVID data pulled from the Ontario API. The second project involved using the MapBox API to present routes on a map calculated from a self created Node.js backend.

App Development

During the 2021 summer I was in an incubator program where my team and I created a React Native app that allowed restaurant customers to pay and order food through their phones. The app is still in progress, however I gained experience creating a lot of front-end elements in React Native and used Expo to build the project on an IOS device.

Game Development

I have experience building games with both Unity and the Unreal Engine. In Unity, I built a 2D top down dungeon crawler with procedurally generated levels, a unique resource mechanic, and complete with a final boss battle. In Unreal I built a 3D tower defence game mimicking Bloons Tower Defence with a unique math quiz element to score points.
